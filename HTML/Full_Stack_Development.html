React

A Javacript library used for building UI and optimized for recording rapidly changing data. Development includes single page-web applications, chart graphics and even 3D development.

Node

A JavaScript runtime environment for executing JavaScript code outside of a browser. The primary backend for a variety of projects, NodeJS was used with Django, React, Express and even Docker applications.

Java

An object-oriented programming langauge that is class-based and designed to have few implementation dependencies. Previously used with Springboot and Thymeleaf, pairing with React and Vue for more advanced web applications has also been practiced.

JavaScript

High-level interpreted scripting language with dynamic typing and protoype-based object-orientation. Primarily used for front end development, JavaScript (and its flavors) provide the modern, interactive feel to web applications.

HTML/CSS

Hypertext Markup Language and Cascading Style Sheets are used in conjuction to design and style documents displayed in a web browser. HTML and CSS have been used in all front end development projects, and are often combined with JavaScript for the most interactive UI.

Django

Python-based web framework designed for database-driven websites that follows MTV arichitecture. Combined with Node, Django has been used to build web applications specifically designed for data manipulation.
