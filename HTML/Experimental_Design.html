Virtual Reality Development

Virtual Reality (VR) is a fully immersive simulated experience that can simulate fantasy or the real world. From creating 3D scatterplots to building room-scale simulated experiences, VR development has been driven in Unity 3D with scripting in C#.

DevOps

Practices that combine software development and IT operations to shorten development life cycle and simultaneously provide continuous delivery. Rigorous practice with version control through Git, virtualization and automation using Docker, and all-level regression VR testing are just some of the DevOps practices performed.

Vertical Scaling

Moving a computing instance to a more powerful machine or increasing the power of a single machine by upgrading existing hardware. Multiple powerful machines have been designed and built for advanced VR development and heavy processing. Use of cloud computing to access additional server/GPU resources in both professional and personal development a regular occurence.

Horizontal Scaling

Partitioning an instance across multiple machines in order to take advantage of an increased pool of resources. Extensive, intensive useage of AWS services such as Lambda, Gateway, API and even IoT have provided a thorough entrance into horizontal scaling.
